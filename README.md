# Jessica's D491 Master Trainer

A pastel WGU-style D491 study trainer with:
- 72-question mock quiz
- Match Cards
- Repeat Misses
- Hidden Keywords
- English + Spanish explanations
- Real scenarios
- Local progress saved in the browser

## Fast GitHub Pages setup

1. Create a new public GitHub repository named `d491-master-trainer`.
2. Upload all files from this folder:
   - `index.html`
   - `style.css`
   - `questions.js`
   - `app.js`
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Click **Save**.
6. Your link will look like:
   `https://Homura13.github.io/d491-master-trainer/`

If the page does not appear immediately, wait 1–2 minutes and refresh.
